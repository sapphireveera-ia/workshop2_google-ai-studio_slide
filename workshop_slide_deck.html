Platform

Apa itu Google AI Studio?

Bukan aplikasi chat biasa. Ini adalah "dapur" tempat developer dan kreator bereksperimen.
Gemini App = Chatting  |  AI Studio = Building

1. Chat & Freeform Prompt

Ruang eksperimen untuk mengatur "System Instructions" agar AI bertindak sesuai persona spesifik yang kita buat.

2. Structured Output

Fitur memaksa model mengeluarkan format data yang rapi (seperti JSON/tabel) dengan memberikan contoh (Few-shot prompting).

3. Pilihan Model Bebas

Akses langsung bereksperimen dengan berbagai model: Gemini Flash (cepat & efisien) hingga Gemini Pro (reasoning tingkat lanjut).

4. Get Code

Prompt yang udah jadi bisa langsung di-export jadi kode siap pakai (Python, JavaScript, atau REST API).

CORE

Seni Prompt Engineering

Merancang instruksi sistematis menggunakan: System Instruction, Context, dan Few-shot Examples.

Prompt Buruk (Terlalu Biasa)

"Buatin ringkasan dari materi PDF jaringan komputer ini buat saya belajar ujian besok."
Kenapa buruk? Tidak ada instruksi format, tidak jelas level bahasanya, dan outputnya bisa jadi sekadar paragraf panjang yang membosankan.

Prompt Baik (Terstruktur & Kontekstual)

[System Instruction] Kamu adalah "Tutor IT Pintar", asisten dosen yang ramah dan ahli menyederhanakan materi rumit.

[Context] Saya mahasiswa smt 3 yang sedang belajar Jarkom, tapi susah fokus membaca teks panjang.

[Task] Buat ringkasan dari teks berikut. Format wajib:

1. 3 Poin utama (gunakan bullet points + emoji).

2. Analogi sederhana (misal: "IP Address itu ibarat rumah...").

3. Berikan 1 kuis PG di akhir.

Teks materi: {Tempelkan teks}

Kenapa baik? Role jelas, ada konteks audiens, batasan format pasti, dan ada interaksi (kuis).

Tinggal Copas, Langsung Jalan! 💙

1. Buka aistudio.google.com → Create New Prompt (Chat Prompt)  •  2. Tempel ini ke kolom System Instruction  •  3. Tempel materi kuliah kamu di kolom chat, klik Run!

💡 Kalau tombol "Salin Prompt" nggak jalan (biasanya kalau file dibuka langsung tanpa hosting), klik tombolnya sekali lagi — teksnya bakal ke-select otomatis, tinggal tekan Ctrl+C.

Contoh materi buat tes cepat (kalau belum punya materi sendiri): "Subnetting adalah proses membagi jaringan besar menjadi beberapa jaringan kecil (subnet) agar lebih efisien dan mudah dikelola."

Setelah dapat hasil JSON-nya, coba ketik "aku masih bingung" di chat — lihat AI-nya masuk ke mode diskusi! Angkat tangan kalau stuck, fasilitator siap bantu.
